NZ Property Valuator — Last updated: 21 February 2026

What this extension does

NZ Property Valuator is a Chrome browser extension that displays property valuation estimates alongside TradeMe property listings. When you visit a supported listing page, the extension extracts the property address from that page and looks it up on four third-party NZ property data services: OneRoof, homes.co.nz, PropertyValue.co.nz, and RealEstate.co.nz.

Data accessed

  • Property address strings only. The extension reads the street address, suburb, and city from the TradeMe listing page you are currently viewing. This is the only data accessed from your browser.
  • No personal information, browsing history, cookies, login credentials, or any other user data is accessed or processed.

How address lookups work

Address queries are sent directly from your browser to the four third-party services listed above. The extension acts as a thin client: it formats the address and makes the same API requests that those services’ own websites make. No address data passes through any server we control.

The privacy policies of those third-party services apply to the requests made to them:

  • OneRoof Privacy Policy
  • homes.co.nz Privacy Policy
  • PropertyValue.co.nz Privacy Policy
  • RealEstate.co.nz Privacy Policy

Caching

Valuation results are cached in memory only, local to the browser session, with a 30-minute expiry. Nothing is written to disk, synced to a server, or persisted beyond the current browser session. The cache is automatically cleared when the extension’s service worker is terminated (e.g., after the browser is closed or the extension is reloaded).

chrome.storage usage

The extension uses chrome.storage.sync to remember which valuation sources you have enabled or disabled in the popup. This preference data is synced by Chrome across your signed-in devices if you have Chrome sync enabled; it contains no personal information, only boolean toggles for each source.

chrome.storage.local is used to cache the most recently fetched estimate and timestamp for each source, so the popup can display a summary without re-fetching. This data is local to your device.

Analytics and tracking

This extension contains no analytics, no telemetry, no crash reporting, and no tracking of any kind. We do not know how often the extension is used, which properties are viewed, or who uses it.

Permissions

The extension requests the following Chrome permissions:

  • storage — to save your source toggle preferences across sessions.
  • host_permissions for TradeMe, OneRoof, homes.co.nz, PropertyValue, and RealEstate — to make the valuation API requests on your behalf.

No other permissions are requested.
